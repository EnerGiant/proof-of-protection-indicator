Proof of Protection Indicator

An interactive working model of the assurance and consequence measures set out in Appendix B of Centripetal Safety (EnerGiant Press). It lets you drive the geometric-mean assurance index, the critical-control gate, and the divergence signal by hand.

A single self-contained HTML page. No build step, no dependencies, no external requests.

Publish with GitHub Pages


Create a repository named proof-of-protection-indicator.
Add index.html and this README, and push to the default branch.
Settings, then Pages, set Source to the default branch, root.
Live at https://<username>.github.io/proof-of-protection-indicator/.


To serve it at energiantpress.com/proof-of-protection-indicator, add the domain under Settings, Pages, and point DNS at GitHub Pages. GitHub writes a CNAME file for you. Whichever address you settle on is the one printed, with its QR, in Appendix B.
